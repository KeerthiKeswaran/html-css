# Task 3: Modern Responsive Product Grid

A premium, responsive product showcase built using modern CSS Grid and pure semantic HTML.

## Objectives
Create a clean, responsive grid layout that elegantly displays multiple product items across different screen sizes.

## Requirements
- **CSS Grid Layout**: Arrange items in **three columns** on desktop screens.
- **Mobile-First Responsiveness**: Automatically adjust to a **single column** on mobile devices using media queries.
- **Consistent Alignment**: Use `gap` and `padding` to ensure uniform spacing and alignment between all items.
- **Pure HTML & CSS**: Built without external frameworks or JavaScript for maximum performance and compatibility.

## Project Overview
This project features a collection of premium product cards, each including:
- High-quality product images with hover-to-zoom effects.
- Clean typography using the **Outfit** Google Font.
- Interactive buttons with hover and active states.
- A modern, minimalist aesthetic with glassmorphism-inspired subtleness.

## Setup Instructions

### Prerequisites
You only need a modern web browser (Chrome, Firefox, Safari, or Edge) to view this project.

### Running Locally
1. **Clone or download** the repository to your local machine.
2. Navigate to the `task-3` directory.
3. Open `index.html` in your favorite web browser.
   - *Alternatively, if you use VS Code, you can use the **Live Server** extension for a real-time preview.*

## Dependencies
- **HTML5**: For structure and semantics.
- **CSS3**: For layout (Grid), styling, and animations.
- **Google Fonts**: Uses the [Outfit](https://fonts.google.com/specimen/Outfit) font family.
- **Local Assets**: Product images are stored in the `./assets/` directory.
