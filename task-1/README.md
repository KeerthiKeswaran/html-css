# Task 1: Personal Portfolio Website

## Overview
This project is a modern, responsive personal portfolio website designed to showcase a professional introduction, experience, projects, and contact information. The objective of this task was to create a clean, aesthetically pleasing, and functional web interface using semantic HTML5 and vanilla CSS3.

## Key Objectives
- **Responsive Hero Section**: A centered, welcoming introduction with proper typography and a clear professional title.
- **Side-by-Side Flex Layout**: A balanced "Contact" and "Quick Links" section using a symmetrical two-column "Flex Column" card layout on desktop view.
- **Desktop Optimization**: Increased content width (1200px) to maximize space on larger screens while maintaining side margins.
- **Mobile First Design**: Fully responsive layout that automatically stacks and scales correctly on small screens (600px and below).
- **Navigation UX**: A sticky navigation bar with a glassmorphism backdrop-filter (frosted glass blur) for a premium user experience.
- **Maintainable Code**: Properly documented and commented HTML and CSS files for scalability and ease of understanding.

## Setup Instructions
1.  Clone or download the repository to your local machine.
2.  Navigate to the `task-1` directory.
3.  Open `index.html` in any modern web browser (Chrome, Firefox, Safari, Edge).
4.  No build step or server is required; however, using a "Live Server" extension (e.g., in VS Code) is recommended for real-time development.

## Dependencies
- **Core**: HTML5, CSS3.
- **Fonts**: Segoe UI, Arial, sans-serif (system defaults).
- **External Libraries**: None (pure Vanilla CSS for maximum performance).
