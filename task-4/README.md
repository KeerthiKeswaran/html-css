# Task 4: Pure CSS Dropdown Menu

## Overview
The objective of this task is to design a premium, responsive navigation bar featuring a multi-level dropdown submenu that appears on hover. The project emphasizes clean code, semantic HTML structure, and smooth CSS transitions to create a high-end user experience.

### Key Features
- **Semantic Structure**: Uses nested `<ul>` elements for menus and submenus.
- **Pure CSS Interactions**: Employs CSS pseudo-classes like `:hover` to trigger submenus and custom transitions for a smooth reveal effect.
- **Premium Aesthetics**: Features a modern "dark mode" design with glassmorphism (backdrop blur) and subtle hover effects.
- **Responsive Design**: Includes a dedicated mobile-first approach with a hamburger menu fallback for smaller screens.

## Setup Instructions
1.  Navigate to the `task-4` directory.
2.  Open `index.html` in any modern web browser (Chrome, Firefox, Edge, or Safari).
3.  Hover over the "Services" or "Solutions" menu items to see the dropdown submenu in action on desktop.
4.  Resize the window or use browser developer tools to view the responsive mobile layout.

## Dependencies & Resources
- **Fonts**: [Google Fonts - Poppins](https://fonts.google.com/specimen/Poppins) (loaded via CDN).
- **Icons**: Simple CSS-only chevron icons (no external icon libraries used).
- **Frameworks**: None. This is a vanilla HTML/CSS project to demonstrate core web development skills.
