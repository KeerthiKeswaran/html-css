# Task 2: CSS Card Component with Hover Effects

## Overview
This task involves building a realistic, professional-grade **debit card UI carousel** using only semantic HTML and modern CSS. The objective is to demonstrate clean layout principles, realistic design mimicry, and smooth interactive hover effects without the use of JavaScript or external libraries.

## Task Objectives
### CSS Card Component with Hover Effects
**Objective**: Build a card that includes an image, a title, and a description.

**Requirements**:
- Style the card with borders, shadows, and padding.
- Add a CSS hover effect (e.g., a background color change or slight scale transformation) with smooth transitions.

## Implemented Result
For this task, we implemented a **premium debit card UI carousel**. Key features include:
- **Semantic HTML**: Using `<section>`, `<article>`, and `<header>` for a meaningful document structure.
- **Realistic Elements**: Each card includes a metallic EMV chip, distinct background textures (Navy and Gold themes), and a professional bank logo.
- **Dynamic Interactions**: A smooth hover transition that scales and lifts the card (`transform: scale(1.03) translateY(-5px)`), creating a high-end banking app feel.

## Setup Instructions
To view this project locally on your machine, follow these steps:

1. **Clone or Download** the repository to your local system.
2. Navigate to the `task-2` directory.
3. Open the `index.html` file in any modern web browser (Chrome, Firefox, Safari, or Edge).
4. Ensure the `assets/` folder contains the necessary image files (`bg-navy.png`, `bg-black-gold.png`, `chip.png`, `bank-logo.png`) for the visual components to render correctly.

## Dependencies
- **None**. This project is built using vanilla HTML5 and CSS3. 
- **Google Fonts**: Uses the [Inter](https://fonts.google.com/specimen/Inter) font family via `@import` in CSS.
